Questo programma e' diviso in 4 diverse categorie:
1. Analisi scraps
2. Analisi fails
3. analisi repair
4. analisi breakdown

ogni sezione esegue diverse query in sql per trovare i dati settimanali, mensili e semestrali e li analizza creando 4 diverse emails che contengono una breve analisi e, se  e'il caso, delle proposte di kaizen projects. 
I kaizen project vengono registrati in sql e sevono per verificare la loro fefficienza.
